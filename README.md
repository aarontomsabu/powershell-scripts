# PowerShell Scripts for IT Administration

This repository contains PowerShell automation scripts commonly used in IT administration and support environments.

## Included Scripts

### UserAccountManagement.ps1
- Creates Active Directory users from a CSV file
- Automates bulk user provisioning
- Reduces manual administrative effort

### DiskCleanup.ps1
- Deletes temporary files from Windows systems
- Helps reclaim disk space
- Supports routine system maintenance

### SystemMonitoring.ps1
- Collects system information
- Exports reports to CSV format
- Assists in monitoring and auditing

## Technologies Used
- PowerShell
- Windows Administration
- Active Directory
- CSV Reporting
